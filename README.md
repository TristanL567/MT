This repository contains the code and analysis for my Master Thesis.

## Project Structure

* **01_Code/**: R scripts for cleaning and analysis.
* **02_Data/**: Raw and processed data (ignored by Git for privacy/size).
* **03_Other/**: Miscellaneous files and notes.
* **Research/**: Literature and reference materials.

## How to Run

1. Open `MT.Rproj` in RStudio.
2. Run the scripts in `01_Code` in numeric order.
